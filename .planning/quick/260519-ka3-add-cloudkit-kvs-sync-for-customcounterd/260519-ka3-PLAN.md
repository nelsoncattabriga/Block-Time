<objective>
Add CloudKit KVS sync for the `CustomCounterDefinition` array so user-defined field definitions roam across devices alongside the rest of the settings.

Purpose: Today only `customCountLabel` (legacy single-counter label) is synced. After the multi-counter rework (260517-u0w / 260519-g0g), each device maintains its own `CustomCounterService.shared.definitions` array independently — definitions added on iPad do not appear on iPhone. This plan closes that gap using the existing KVS pipeline.

Output: Two-file change — `CustomCounterService` gains a `replaceAll(_:)` mutator; `CloudKitSettingsSyncService` encodes/decodes the definitions array as JSON in KVS and reconciles on download.
</objective>
